PhoneGap
========

PhoneGap is a web platform that exposes native mobile device apis and data to JavaScript. Allows you to build cross-platform mobile applications using HTML, JavaScript and CSS.

### PhoneGap is a distribution of Apache Cordova. It will always remain free and open source.

---

More information on Apache Incubator Cordova Project and the incubation process can be found here:
[http://incubator.apache.org/projects/callback.html](http://incubator.apache.org/projects/callback.html)

Directory Structure
-------------------

	  |-doc/ ........... source documentation
	  |-lib/ ........... platform code for supported operating systems
	  | |-android/
	  | |-bada/
	  | |-blackberry/
	  | |-ios/
	  | |-symbian/
	  | |-webos/
	  | '-windows/
	  |-changelog ..... a changelog compiled from comments and authors
	  |-license ....... the Apache Software License v2
	  |-version ....... release version in plain text
	  '-readme.md ..... release readme
